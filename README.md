# Direct Chat for Flarum

![Flarum Extension](https://img.shields.io/badge/Flarum-Extension-e74c3c?style=for-the-badge)
![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)

Flarum forum siteleri için anlık, şık ve modern bir mesajlaşma (Direct Messaging) eklentisi.

## 🚀 Özellikler

*   **Premium Tasarım:** Sağ altta açılan, modern ve kullanıcı dostu mesajlaşma arayüzü.
*   **Profil Entegrasyonu:** Kullanıcıların profillerinde "Mesaj Gönder" butonu.
*   **Anlık İletişim:** Sayfa yenilemeden mesajlaşma desteği (Flarum API tabanlı).
*   **Çoklu Dil Desteği:** Türkçe ve İngilizce dilleri tam uyumlu.
*   **Güvenli Altyapı:** Sadece giriş yapmış kullanıcılar arasında mesajlaşma kontrolü.

## 🛠️ Kurulum & Güncelleme

Eklentiyi Composer aracılığıyla forumunuza dahil edebilirsiniz:

```bash
composer require framiodev/direct-chat
```

Eklentiyi son sürüme güncellemek için:

```bash
composer update framiodev/direct-chat
php flarum cache:clear
```

Ardından forumunuzun yönetim panelinden (Admin Dashboard) eklentiyi aktif edin.

## ⚙️ Gereksinimler

*   Flarum v1.8.0 veya üzeri.
*   PHP 8.0+
